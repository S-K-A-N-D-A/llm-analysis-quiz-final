# LLM Quiz Evaluation API

This repository provides documentation for a quiz evaluation API built using Supabase Edge Functions via the Lovable platform. The API is used to accept quiz tasks sent from an external evaluation system and respond with outputs generated by an LLM.

## Purpose
This API receives a POST request containing a quiz question or task and returns an evaluated result. It is used for automated model analysis and testing.

## API Endpoint
POST requests should be sent to:
https://qbzjakjzpkqkdodnwcxq.supabase.co/functions/v1/quiz
